UI Design Process

Designing the Interface for digital products are differ with designing visual for print product. RGB and CMYK really are'nt in the same way, also there are some gap layout between digital and print product that difficult to do. Usually i designing from scratch, moodboard idea for refrence, brainstrom concept and sketching, decide and then design. I used these tools for work : Sketchapp, Figma, Gravit Designer, Photoshop, Illustrator, manual drawing.

UX Design Process

Digital product without user on it are big fail. user must be on the center and base guide to build the experience in product for win the competition. Research techniques needed to collect user persona data to build near best product for had rich user experience. Design thinking like Sprint design mostly used among team player. Designing UX start from define the problem, brainstroming to solve it, & validation the solution. Tools for work : Sketchapp, Figma, Adobe Xd, Invision

Brand Design

Brand design is about the visual image that bring unique character in the product as differential point from others, easly to remember and stand out. In the other hand brand design can be face of the product to the user. Branding design include logo, color, shape, font and if needed mascot / illustration could be the perfect communication to the user easly. Every brand design had their own story to tell through the visual not the verbal. Tools for work : depend on creativity anything could be

Website Design

Website is online media that best for reach people everywhere using internet connection. Web responsif are the best layout for fill various screens size. i was learning some UI framework to designing website such as Materialize, Bootstrap, and AngularJS, also learning how the code HTML5, CSS3, JS works. The process to build apps didnt differ with build website, from Architectur Information, flowchart, wireframe, UI/UX, code.
